Play About Rules Guide Manual Commands FAQ

Commands and Mechanics

The entire Command & Conquest system relies on the use of the Shard bot and the #command_and_conquest channel found on the Thegye Discord. All commands and mechanics are listed here for ease of reference. Review the Guide and Manual for additional information.

Note: all commands come with complete descriptions within Discord. You can view all the C&C related commands by typing /cnc. Most commands are not usable in DMs.

Base Commands

/cnc register [nation name] [color]: this command registers you as a new user. The first requirement is the nation name variable, which should be a creative and interesting name. It should not include a prefix, as that is set by the Government Subtype of the nation. The color variable should be a hex code (e.g., #481274). If the color provided is too close to a taken or reserved color, the register command will fail.

/cnc info: this command sends the current information about the entire Command & Conquest system.

/cnc turn: this command sends the current turn, as well as when the next turn update is in your timezone.

/cnc view map: this command opens the map menu, which provides four map modes: Main (which displays the base map), Terrain (which displays the terrain without numbers), Geography (which displays the names of locations), and Nation (which displays the owned/occupied territory of all nations). Click each button to display each mode.

/cnc view locate_province [province id]: this command generates a map file highlighting the specific province provided in a bright pink color to help users locate a specific province on the map.

/cnc view market_values: this command displays the current market values for each of the 30 Trade Goods produced across the map.

/cnc view good_production: this command displays the current total production of the 30 Trade Goods.

Nation and Province Commands

/cnc nation (nation name OR user): this command displays information about the nation or user selected. If the inputs are left blank, it will show information about your nation. Calling other nations will display two buttons: Cooperative Relations and Hostile Relations. Each will pull up the corresponding menu. Cooperative Relations will display four base buttons: Diplomatic Relations, Military Alliance, Trade Pact, and Military Access. The Propose Subjugation and Manage Puppet buttons may display if applicable. Hostile Relations will display two buttons: Embargo and War.

/cnc dossier: this command privately displays detailed information about your nation using six buttons: Summary, Authority, Military, Economy, Government, and Relations. Click each button to display each menu.

/cnc strategic_view: this command privately displays detailed information about all of your provinces.

/cnc province [province id]: this command displays information about the given province. The status of the province will open different button menus. Provinces owned by you will display five buttonds: Construct, Deconstruct, Boost Development, Appropriate Development, and Abandon Province. Provinces owned by others will display no menu. Provinces owned by natives will display the Colonize button if you Researched the Colonization technology.

Army and General Commands

/cnc army [army id]: this command displays information about the specified army. Armies that are owned by you will appear with equal signs in the suggested list. Armies owned by you display four buttons: Recruit, Disband, Assign General, and Embark/Disembark, if you have Researched the Sailing technology. Use the Assign General button to recruit and reassign Generals.

/cnc general [general id]: this command displays information about the specified general.

/cnc create_army [posting] (recruit_troops): this command creates a new army in the specified location. You can optionally add a number of troops to recruit directly into the army.

/cnc move_army [army id] [move_to]: this command will move the specified army from its current location to the specified location, if it has enough movement. If it is blocked by an enemy army, fort, or terrain, it will either pathfind around the obstacle or fail to move. If an enemy army or armies are in the location, a Battle will initiate. Embarked armies will attempt to move by water.

/cnc army_report: this command will privately display information about all of your armies.

War Commands

/cnc view wars (view_all): this command will display all wars your nation is currently engaged in. Selecting "True" for view_all will display all ongoing wars, regardless of whether you are involved.

/cnc war_info [war id]: this command will display information about the specified war.

Technology and Research Commands

/cnc tech [tech]: this command displays information about the specified technology, including prerequisites and effects.

/cnc research [tech]: this command begins researching the specified technology.

\

/cnc researching: this command displays the current research. Selecting the Cancel button will end any current research.

Government Commands

/cnc government: this command opens the Government Management menu with four buttons: Taxation (for managing your tax level), Public Spending (for managing current public spending), Military Upkeep (for managing current military upkeep), and Boost Stability (for increasing current stability).

/cnc reform_government: this command opens the Government Reform menu with three buttons: Reform Government Type (for changing your overall government type), Reform Government Subtype (for changing your specific government subtype), and View Governments (for browsing the different Government Types and Subtypes).

/cnc designate_capital [province id]: this command establishes the national capital in the specified province.

Great Power Commands

/cnc gp_score: this command displays your nation's current Great Power score calculation.

/cnc great_powers: this command displays all current nations in order of Great Power score.
